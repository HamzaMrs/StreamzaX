## StreamzaX

### Lancer le projet en local

1. Installez les dépendances :
   ```bash
   npm install
   ```

2. Démarrez le serveur de développement :
   ```bash
   npm run dev
   ```

3. Ouvrez le site dans votre navigateur à l'adresse indiquée (par défaut http://localhost:5173/ ou http://localhost:5174/).
